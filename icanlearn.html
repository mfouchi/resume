I CAN Learn Educational Software

Role : Founding member, Sr. Project Manager, Sr. Software Engineer
Front End : ASP, HTML/CSS, Javascript, Flash
Business Logic : Visual Basic COM, .NET Web Services
Back End : SQL Server 2000/2005, Amazon EC2, S3
Host : Private IIS Servers, Amazon EC2
Tools : Visual Studio, SQL Server Management Studio, SQL Profiler, VS Sourcesafe

All students in a classroom would wear headphones at their own computer and progress through courseware at their own pace. Each video lesson would present a 'pretest' that would allow the student to move on to the next lesson if they passed. Consequently, students were not held back by slower students, or left behind by faster students. The the teacher was free to work with individual students who needed help. The teacher could monitor and manage every student's progress from their 'Classroom Explorer'. This company is now closed.
