XactRate

Role : Designer, Project Manager
Front End : ASP.NET MVC
Back End : Azure SQL
Host : Microsoft Azure
Tools : PhoneGap, Sencha Touch
Website : www.XactRate.com

Project is released and currently in use by some HVAC companies, but never gained traction. Business owners setup company metrics to define their product markup and labor hour charges. These numbers change dynamically as business progresses based on actual service calls / income. Tools include invoicing, heat load calculator, service call scheduling, time tracking, inventory, and product catalogs.
